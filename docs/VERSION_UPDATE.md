# 版本更新说明 - v3.0 (统一版)

## 📅 更新时间
2026-03-09

## 🎯 更新概述

将基础版 (`main.py`) 和增强版 (`main_enhanced.py`) 统一为单一版本，简化项目结构，提升用户体验。

---

## ✨ 主要变更

### 1. 代码统一
- ✅ **增强版完全替代基础版**
  - `main_enhanced.py` → `main.py`
  - 保留旧版代码作为参考：`main_backup.py`
  
- ✅ **功能全面升级**
  - 保留所有基础版功能
  - 新增集成学习、SMOTE、特征工程等高级功能

### 2. 依赖简化
- ✅ **统一依赖文件**
  - 删除 `requirements_minimal.txt`
  - 删除 `requirements_enhanced.txt`
  - 创建 `requirements.txt`（包含所有依赖）

### 3. 文档更新
- ✅ **README.md 全面重写**
  - 移除基础版/增强版的区分
  - 统一为一个版本的说明
  - 更新项目结构
  - 更新使用指南

### 4. 文件结构优化
```
优化前：
├── src/
│   ├── main.py                # 基础版
│   └── main_enhanced.py       # 增强版
├── requirements_minimal.txt
└── requirements_enhanced.txt

优化后：
├── src/
│   ├── main.py                # 统一版本 ⭐
│   └── main_backup.py         # 旧版参考
├── requirements.txt            # 统一依赖
└── docs/                      # 文档目录
```

### 基础版功能（全部保留）
- ✅ 数据增强（随机删除、随机交换）
- ✅ 类别权重处理
- ✅ 交叉验证（3 折）

  - SVM

  - TF-IDF (1000 维)
  - 13 维文本统计特征

  - SMOTE 过采样
- ✅ **评估优化**
  - 5 折分层交叉验证
  - 阈值优化
  - AUC 指标
