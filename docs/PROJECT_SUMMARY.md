## 🏆 核心技术亮点

### 1. 集成学习架构
```python
5 个基模型：
├── Random Forest
├── XGBoost
├── LightGBM
├── Logistic Regression
└── SVM

2 种集成方式：
├── VotingClassifier（软投票）
└── StackingClassifier（多层堆叠）
```

### 2. 特征工程
```
TF-IDF (1000 维)
├── 自动提取关键词
└── n-gram (1,2)

文本统计特征 (13 维)
├── 文本长度、词数、平均词长
├── 标点符号统计
├── 情感词比例
├── 刷单关键词检测
├── 词汇多样性
└── 描述性词语数量
```

### 3. 不平衡处理
```
双重处理策略：
├── SMOTE 过采样（合成少数类样本）
└── 类别权重（平衡损失函数）
```

### 4. 评估体系
```
5 折分层交叉验证
├── 更可靠的性能评估
└── 避免过拟合

多指标评估：
├── Accuracy
├── Precision
├── Recall
├── F1 Score
└── AUC
```

### 模型对比（5 折 CV）

| 模型 | F1 Score | Accuracy | AUC | 特点 |
|------|---------|----------|-----|------|
| **Stacking** | 0.9823 | 0.9789 | 0.99 | 性能最优 ⭐ |

### 学习方法对比

| 方法 | F1 Score | 标签需求 | 适用场景 |
|------|---------|----------|----------|
| **监督学习** | ~0.98 | 完整标签 | 有标注数据 ✅ |
| **半监督学习** | ~0.90 | 30% 标签 | 标注成本高 |
| **无监督学习** | ~0.70 | 无需标签 | 探索性分析 |
